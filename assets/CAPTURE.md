# Screenshots for the store README

Drop PNGs into **this folder** (`assets/`) with the exact filenames below and the
README's images light up. Until then, GitHub/Obsidian show broken-image placeholders.

Capture on a retina display, crop tight; **light theme reads best** in the store.
Use a **demo/sample vault** — never show private notes. Keep each image under ~1 MB,
≤1600px wide; PNG is fine for UI.

## Required (the README references all five)

| Filename | What to shoot |
|---|---|
| `chat-panel.png` | **Hero.** The Companion side-panel chat: a message with **Context** chips attached (active note / linked notes), mid-stream or with a completed reply. |
| `artifact-inline.png` | A `claude-html` artifact rendered **inline** in a note (the sandboxed iframe) — a dashboard or report in the clay/olive palette. The signature feature. |
| `session-to-note.png` | A **captured Claude Code session** open in Obsidian: frontmatter (`claude-session`, model, branch, tokens) + the distilled conversation + "What Claude did" + files touched. The new 0.6.0 feature. |
| `manifest-roadmap.png` | A `manifest-pm` / `manifest-vault` roadmap or health artifact — the ranked, prioritized output. |
| `chat-controls.png` | The in-chat model switcher + extended-thinking effort dial (+ reasoning panel). |
